#SuperMarket PoS
This is part of a project for IUSB college course C311 - Programming Languages

Originally made to be in Kotlin as a mobile app. Changed course and decided to utilize javascript, css, and html. This will be a point of service site, this site will be recording sales, managing invoices, and marking daily sales and sales goals. I will be adding in seperate pages for each of the major needs, i.e. Backstock, FillFloor,IncomingInventory, Inventory, and SalesPage.
